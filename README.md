# housing-price-nn
PyTorchを用いた住宅賃料予測のためのニューラルネットワーク実装。物件の属性データ（築年数、面積、設備等）から賃料を推定する多層パーセプトロンの学習・評価コードです。


# 住宅賃料予測モデル (Rent Prediction Model)

本リポジトリは、研究で使用した住宅賃料予測のためのニューラルネットワーク（多層パーセプトロン）の実装および評価コードを公開するものです。

## 概要
物件の属性データ（築年数、専有面積、設備数、所在地など）を入力とし、対象物件の賃料を推定するモデルを構築しています。データの基礎集計から、機械学習に向けた高度な前処理、PyTorchを用いた深層学習モデルの訓練・評価までの一連のパイプラインを含んでいます。

## ファイル構成と役割

本リポジトリは、以下の3つの主要なPythonスクリプトで構成されています。上から順に実行することで、生データから最終的な予測モデルの構築までを行うことができます。

### 1. 01_data_analysis.py（基礎集計と可視化）
- 賃貸の生データを読み込み、目的変数（賃料）の基本統計量（平均、中央値、最頻値、標準偏差など）を算出します。
- 賃料の分布状況を把握するためのヒストグラムを自動生成し、画像ファイルとして保存します。

### 2. 02_data_preprocessing.py（データの前処理）
- 機械学習モデルに入力するためのデータ整形を行います。
- 主な処理内容:
  - 設備関連の列からの「設備数」の自動カウント
  - 建物構造や所在地に対するOne-hotエンコーディング
  - 間取りタイプに対するターゲットエンコーディング（平均値・中央値）
  - 共益費や徒歩分数などの欠損値のゼロ埋め補完
  - カテゴリ変数のラベルエンコーディング

### 3. 03_train_nn.py（ニューラルネットワークの学習と評価）
- 前処理済みのデータを読み込み、PyTorchを用いて多層パーセプトロン（MLP）による回帰モデルを訓練します。
- 主な機能:
  - StandardScalerによる特徴量とターゲットの標準化
  - 過去の学習済みモデル（.pth）からの学習再開（チェックポイント機能）
  - 訓練データとテストデータの誤差（MAE）比較による汎化性能（過学習）の検証
  - 損失関数（Loss）の推移、予測値vs実際値の散布図、ヒートマップ等の評価グラフの自動生成と一括保存機能

### その他
- requirements.txt : 実行環境のライブラリ依存関係一覧

## 実行手順（使い方）
1. 対象となるExcelデータ（生データ）を用意します。
2. 01_data_analysis.py を実行し、データの傾向を把握します。
3. 02_data_preprocessing.py を実行し、機械学習用の前処理済みデータを生成します。
4. 03_train_nn.py を実行し、ニューラルネットワークの学習と評価を行います。結果は日時付きの専用出力フォルダ（neural_network_outputs_...）に自動保存されます。

## 免責事項・補足
本プログラムでは結果の再現性を確保するために乱数のシード値を固定しています。ただし、実行するハードウェア環境（CPUやGPUのアーキテクチャ）やOS、各種ライブラリのわずかなバージョンの違いによって内部の計算精度に影響が出ることがあるため、お手元の環境で実行した際に最終的な学習結果や誤差に微小な差異が生じる場合がある点にご留意ください。
